window.KarmaOyunlar GLOBAL NESNESİ (alt oyun sayfaları MUTLAKA buna uymalı):
- js/main.js yalnızca ana sayfada (index.html) yüklüdür. Alt sayfalar (mayintarlasi/index.html) kendi script'lerini barındırır AMA aynı localStorage anahtarını ve tema mekanizmasını PAYLAŞMALIDIR.
- DİKKAT: window.KarmaOyunlar SADECE ana sayfada vardır. Alt sayfalar bu nesneye GÜVENMEMELİ; onun yerine aşağıdaki ortak sözleşmeyi kendileri uygulamalı (aynı STORAGE_KEY ve aynı tema/anim mantığı).

ORTAK SÖZLEŞME (alt sayfalar birebir uymalı):
- localStorage anahtarı: "karmaOyunlarAyarlar"
- JSON şeması: { tema: 'dark'|'light', animasyonHizi: 'yavas'|'normal'|'hizli', sesEfektleri: boolean }
- Varsayılan: { tema:'dark', animasyonHizi:'normal', sesEfektleri:true }
- TEMA: document.documentElement.setAttribute('data-tema', 'dark'|'light')  → class DEĞİL attribute. (CSS html[data-tema="..."] seçicilerine bağlı.)
- ANIM: document.documentElement.style.setProperty('--anim-speed', deger) → yavas='0.6s', normal='0.3s', hizli='0.12s'.
- SES: document.documentElement.setAttribute('data-ses', 'acik'|'kapali') (opsiyonel görsel/işlevsel kullanım); asıl tercih ayarlar.sesEfektleri (boolean).
- FOUC önlemek için alt sayfaların da <head>'inde inline script ile localStorage okuyup data-tema'yı erkenden set etmesi önerilir (ana sayfadaki gibi).

js/main.js'in DIŞA AÇTIĞI API (ana sayfa içinde erişilebilir; alt sayfalar referans için):
- window.KarmaOyunlar.ayarlar            → canlı ayar nesnesi {tema, animasyonHizi, sesEfektleri}
- window.KarmaOyunlar.STORAGE_KEY        → "karmaOyunlarAyarlar"
- window.KarmaOyunlar.VARSAYILAN         → varsayılan ayar kopyası
- window.KarmaOyunlar.ANIM_HIZ_DEGER     → { yavas:'0.6s', normal:'0.3s', hizli:'0.12s' }
- window.KarmaOyunlar.oku()              → localStorage'dan okur, doğrular, ayarlar'a yazar, döndürür
- window.KarmaOyunlar.kaydet()           → ayarlar'ı localStorage'a yazar
- window.KarmaOyunlar.getAyar(anahtar)   → anahtar verilirse o değeri, verilmezse tüm ayar kopyasını döndürür
- window.KarmaOyunlar.setAyar(anahtar, deger) → doğrular, uygular, kaydeder, formu senkronlar; başarısızsa false
- window.KarmaOyunlar.temaUygula()       → data-tema attribute'unu uygular
- window.KarmaOyunlar.animUygula()       → --anim-speed CSS değişkenini uygular
- window.KarmaOyunlar.sesUygula()        → data-ses attribute'unu uygular
- window.KarmaOyunlar.tumunuUygula()     → tema+anim+ses topluca uygular
